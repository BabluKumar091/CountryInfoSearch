# 🌍 Country Info Search

A modern and responsive web application built using **React.js** that allows users to explore detailed information about countries worldwide in a fast and interactive way.

## 📌 Overview

Country Info Search is a user-friendly web app where users can search for any country and get detailed information such as population, region, capital, and flag. The application focuses on performance, clean UI, and smooth user experience.

---

## ✨ Features

- 🔍 Real-time country search
- 🌐 Detailed country information view
- 🗺️ Region-based filtering
- 🌗 Light/Dark mode toggle
- 📄 Dynamic routing for country pages (`/:country`)
- ⚡ Fast and responsive UI
- ❌ Custom 404 error page
- 📱 Fully responsive (mobile + desktop)

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Routing:** React Router DOM  
- **Styling:** Tailwind CSS  
- **API:** REST Countries API
